Emplacement Géographique DREN , CISCO , ZAP , FOKONTANY , ETABLISSEMENT des PERSONNES ayant des liens web html vers leurs PORTFOLIOS et dont ces PERSONNES sont affectées à des ETABLISSEMENTS. En résumé ces ETABLISSEMENTS contiennent les LIENS WEB HTML vers les PORTFOLIOS de ces PERSONNES qui sont afféctées à ces ETABLISSEMENTS
